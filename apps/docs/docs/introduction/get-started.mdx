## Try now

Go to [drawink.app](https://drawink.app) to start sketching.

## How are these docs structured

These docs are focused on developers, and structured in the following way:

- [Introduction](/docs/) — development setup and introduction.
- [@drawink/drawink](/docs/@drawink/drawink/installation) — docs for the npm package to help you integrate Drawink into your own app.
- Editor — IN PROGRESS. Docs describing the internals of the Drawink editor to help in contributing to the codebase.
- [@drawink/mermaid-to-drawink](/docs/@drawink/mermaid-to-drawink/installation) - Docs for the mermaid to drawink parser
