# JSON Schema

The Drawink data format uses plaintext JSON.

## Drawink files

When saving an Drawink scene locally to a file, the JSON file (`.drawink`) is using the below format.

### Attributes

| Attribute | Description | Value |
| --- | --- | --- |
| `type` | The type of the Drawink schema | `"drawink"` |
| `version` | The version of the Drawink schema | number |
| `source` | The source URL of the Drawink application | `"https://drawink.com"` |
| `elements` | An array of objects representing drawink elements on canvas | Array containing drawink element objects |
| `appState` | Additional application state/configuration | Object containing application state properties |
| `files` | Data for drawink `image` elements | Object containing image data |

### JSON Schema example

```json
{
  // schema information
  "type": "drawink",
  "version": 2,
  "source": "https://drawink.com",

  // elements on canvas
  "elements": [
    // example element
    {
      "id": "pologsyG-tAraPgiN9xP9b",
      "type": "rectangle",
      "x": 928,
      "y": 319,
      "width": 134,
      "height": 90
      /* ...other element properties */
    }
    /* other elements */
  ],

  // editor state (canvas config, preferences, ...)
  "appState": {
    "gridSize": 20,
    "viewBackgroundColor": "#ffffff"
  },

  // files data for "image" elements, using format `{ [fileId]: fileData }`
  "files": {
    // example of an image data object
    "3cebd7720911620a3938ce77243696149da03861": {
      "mimeType": "image/png",
      "id": "3cebd7720911620a3938c.77243626149da03861",
      "dataURL": "data:image/png;base64,iVBORWOKGgoAAAANSUhEUgA=",
      "created": 1690295874454,
      "lastRetrieved": 1690295874454
    }
    /* ...other image data objects */
  }
}
```

## Drawink clipboard format

When copying selected drawink elements to clipboard, the JSON schema is similar to `.drawink` format, except it differs in attributes.

### Attributes

| Attribute | Description | Example Value |
| --- | --- | --- |
| `type` | The type of the Drawink document. | "drawink/clipboard" |
| `elements` | An array of objects representing drawink elements on canvas. | Array containing drawink element objects (see example below) |
| `files` | Data for drawink `image` elements. | Object containing image data |
